# aiub-notifier
This is a program that emails you the latest AIUB notice. It is
designed to run in the background on your computer or on a hosting
service such as Netlify, Render, or Vercel. 

You need nodejs for this program to run. Create a Google App Password
and copy it to your environment variables (`.env`). **Do not share this 
with anyone**. Refer to `.env.example`.

Run the program with `pnpm build && pnpm start`.
